# Mukesh Thatipamula - Data Analyst Portfolio

GitHub Pages ready portfolio website.

## Upload Instructions

Upload all files from this folder to your GitHub repository root.

Your repository root must show:

```
index.html
MUKESH_Portfolio.html
assets/
README.md
.gitignore
LICENSE
```

## GitHub Pages Settings

Settings → Pages → Deploy from branch → main → /root → Save
